Security Checks Performed

  • Hardcoded Secrets: API keys, passwords, tokens in code
  • SQL Injection: String concatenation in queries, unsanitized input
  • XSS Vulnerabilities: Unescaped user input in HTML/templates
  • File Operations: Path traversal, unsafe file handling
  • Command Injection: Unsafe shell command execution
  • Dependency Issues: Known vulnerable packages
# .git/hooks/pre-commit import ollama diff = get_staged_diff() response = ollama.chat( model='qwen2.5-coder:32b', messages=[{ 'role': 'user', 'content': f'Scan for security issues: {diff}' }] ) if 'VULNERABLE' in response: print("Security issue found!") exit(1) # Block commit
